# UX Copy Review Review Checklist

Use this checklist while executing the skill.

## Preparation

- [ ] Confirm screenshots, component code, content inventory, or target flow.
- [ ] Confirm audience, product voice, terminology, and localization constraints.
- [ ] Confirm known confusing labels, errors, support tickets, or analytics signals.
- [ ] Confirm legal, compliance, brand, or product constraints.

## Execution

- [ ] Inventory all user-facing strings in the target scope.
- [ ] Review each string for clarity, specificity, actionability, tone, consistency, and accessibility.
- [ ] Check labels and CTAs against user intent and expected outcome.
- [ ] Rewrite errors and helper text to explain cause, next step, and recovery without blame.
- [ ] Flag strings that require product, legal, or localization review.

## Completion

- [ ] Copy helps users decide, act, and recover with confidence.
- [ ] Terms are consistent across the reviewed flow.
- [ ] Errors are specific and actionable.
- [ ] Recommendations respect accessibility, localization, and product constraints.
