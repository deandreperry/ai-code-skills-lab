# UX Copy Review

## Skill Name

UX Copy Review

## Description

Review product microcopy including labels, CTAs, helper text, errors, empty states, confirmations, and onboarding messages.

## When To Use

- Interface copy feels vague, inconsistent, too technical, or misaligned with user intent.
- A feature needs clearer labels, errors, CTAs, or instructional content.
- Design and engineering need copy that supports accessibility and localization.
- Users misunderstand what actions do or how to recover from errors.

## Inputs Required

- Screenshots, component code, content inventory, or target flow.
- Audience, product voice, terminology, and localization constraints.
- Known confusing labels, errors, support tickets, or analytics signals.
- Legal, compliance, brand, or product constraints.

## Step-by-Step Workflow

1. Inventory all user-facing strings in the target scope.
2. Review each string for clarity, specificity, actionability, tone, consistency, and accessibility.
3. Check labels and CTAs against user intent and expected outcome.
4. Rewrite errors and helper text to explain cause, next step, and recovery without blame.
5. Flag strings that require product, legal, or localization review.

## Output Format

- Copy review table with current copy, issue, recommendation, and rationale.
- Rewritten labels, CTAs, helper text, errors, and state messages.
- Terminology and localization notes when relevant.

## Success Criteria

- Copy helps users decide, act, and recover with confidence.
- Terms are consistent across the reviewed flow.
- Errors are specific and actionable.
- Recommendations respect accessibility, localization, and product constraints.

## Common Mistakes

- Making copy clever at the expense of clarity.
- Changing legally sensitive wording without review.
- Using internal system terms users do not know.
- Writing errors that describe failure without a recovery path.

## Example Usage

Use this skill to improve form labels, payment error messages, confirmation dialogs, and upgrade CTAs in a billing settings flow.
