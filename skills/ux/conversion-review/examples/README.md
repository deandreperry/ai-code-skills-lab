# Conversion Review Examples

## Sample Prompt

```text
Use the Conversion Review skill.
Inputs:
- Target conversion flow, screenshots, analytics, funnel data, or URL.
- Audience, offer, conversion goal, and traffic source.
- Known objections, support feedback, and competitive context.
```

## Example Scenario

Use this skill to review a SaaS pricing and trial signup flow for clarity, plan comparison, objections, field friction, and analytics gaps.

## Expected Output

- Conversion review with prioritized friction points and recommendations.
- Experiment ideas with hypothesis, target metric, and implementation notes.
- Analytics and instrumentation gaps.
