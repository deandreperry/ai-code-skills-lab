# Heuristic Evaluation Examples

## Sample Prompt

```text
Use the Heuristic Evaluation skill.
Inputs:
- Target interface, prototype, screenshots, or flow description.
- User goals, domain context, and priority tasks.
- Chosen heuristic framework or preference for Nielsen-style heuristics.
```

## Example Scenario

Use this skill to evaluate a project creation wizard against visibility of status, error prevention, recognition over recall, and user control.

## Expected Output

- Heuristic evaluation matrix with issue, heuristic, severity, evidence, and recommendation.
- Top findings summary for stakeholders.
- Validation plan for issues that require user testing or data.
