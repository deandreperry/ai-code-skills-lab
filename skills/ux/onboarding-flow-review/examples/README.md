# Onboarding Flow Review Examples

## Sample Prompt

```text
Use the Onboarding Flow Review skill.
Inputs:
- Onboarding screens, routes, prototype, analytics, or screenshots.
- Activation goal, primary user segment, and success metric.
- Required setup steps, optional steps, and data dependencies.
```

## Example Scenario

Use this skill to review a SaaS onboarding flow that asks users to create a workspace, invite teammates, choose templates, and connect integrations.

## Expected Output

- Onboarding flow assessment with friction points and recommended changes.
- Step-by-step journey map with user goal, system need, and risk at each step.
- Measurement plan for activation, completion, and drop-off analysis.
