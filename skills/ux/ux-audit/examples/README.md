# UX Audit Examples

## Sample Prompt

```text
Use the UX Audit skill.
Inputs:
- Target flow, screenshots, prototype, URL, or product description.
- Primary users, goals, constraints, and success metrics.
- Known pain points, analytics, support tickets, or usability feedback.
```

## Example Scenario

Use this skill to audit a dashboard onboarding flow for clarity, empty states, call-to-action hierarchy, and error recovery.

## Expected Output

- UX audit report with severity, evidence, and recommendations.
- Prioritized issue list mapped to user goals.
- Suggested experiments, instrumentation, or research follow-up when evidence is incomplete.
