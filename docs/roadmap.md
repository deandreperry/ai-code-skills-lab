# Roadmap

AI Code Skills Lab is organized as a product-quality catalog. The roadmap focuses on depth, validation, and adoption rather than sheer skill count.

## Near Term

- Add richer examples for each skill, including before and after outputs.
- Expand resource checklists for accessibility, testing, and design system governance.
- Add category-level indexes and cross-skill recommendations.
- Improve validation for section quality, minimum examples, and duplicate skill names.

## Mid Term

- Add optional TypeScript helper scripts for selected skills such as token extraction and checklist generation.
- Publish integration notes for common AI coding agent environments.
- Add sample project walkthroughs showing multiple skills working together.
- Introduce lightweight skill maturity levels such as draft, ready, and field-tested.

## Long Term

- Build a searchable catalog website.
- Add organization-specific overlays for internal standards.
- Provide commercial support packages for engineering teams that want custom skills.
- Add metrics for skill usage, outcomes, and maintenance health.

## Principles

- Expand only when new skills are executable and valuable.
- Prefer better workflows over more files.
- Keep accessibility, maintainability, and documentation quality central to the product.
